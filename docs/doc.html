    http://api.erp.sodalife.xyz/v1

    Sodalife ERP API Doc

    Resume ¶

    This is an api doc for Soda Systerm

    • 系统架构

      • 本API Server使用GO 1.8.3 编写， 基于iris v5框架开发，使用 JWT 验证，除 Common 类外，所有 API 调用都需提供 Authorization 请求头，附带服务器颁发的 token。登录成功后将返回 token。

      • 所有请求依赖于 Cookie ，注意不要禁用，以及设置 XMLHttpRequest 的 WithCredentials 属性为 true。
